# Form Builder

A simple in-memory form builder built with React, TypeScript, and MUI.

## Features

- View a list of created forms
- Create, edit, and delete forms
- Add multiple fields to a form — each with a name, label, type (`string`, `number`, `boolean`), and required flag
- Live preview of the form as you build it

## Stack

- React 19 + TypeScript
- Vite
- MUI (Material UI)
- Vitest + Testing Library

## Getting started

```bash
npm install
npm run dev
```

## Tests

```bash
npm test
```
